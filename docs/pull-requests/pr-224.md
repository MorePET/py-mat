## Relation to #219

#219 (`test_substrate_repros.py`) covers `mat-vis#313` at the **substrate / live-network** level — `Vis(...).to_threejs()` must not raise against the real CDN. This PR adds **pymat-surface unit-level** coverage using mocked `MatVisClient` instances, so:

- regressions in pymat's tier-resolution / repr / scalars accessor logic surface without network
